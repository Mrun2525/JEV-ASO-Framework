# Figures

## Overview

This directory contains publication-quality figures generated during the computational analyses described in the manuscript.

Figures correspond to the main and supplementary figures included in the publication.

## Typical Contents

- Genome-wide hotspot analysis
- Thermodynamic interaction plots
- RNA accessibility profiles
- Conservation analyses
- Secondary structure visualizations
- Host selectivity analyses
- Integrated prioritization workflow

The original editable files used to generate publication figures should be archived whenever possible.
